# Career Summary

손기석은 **Java/Spring 기반 백엔드 개발 경험**, **Python/FastAPI 기반 API 및 AI 응용서비스 구현 경험**, 그리고 **정보보안 경력**을 함께 보유한 엔지니어입니다.

공공 및 데이터 분석 서비스 프로젝트에서 웹 애플리케이션, API 서버, DB 연계, 운영 반영, 배포 구조를 다뤄왔고, 최근에는 **LLM 챗봇 PoC**, **Text2SQL**, **문서 기반 AI 서비스 구조 검토**, **GPU 서버/Docker 기반 AI 개발환경 구성** 등 백엔드와 AI 응용개발이 결합된 업무를 수행했습니다.

단순 기능 구현만이 아니라,
- 데이터 분석 결과를 실제 서비스 기능으로 연결하고
- 운영 가능한 개발 구조를 만들고
- 협업 체계와 실행 환경을 정리하며
- 필요할 때는 보안·권한·감사 관점까지 함께 고려하는

실무형 엔지니어를 지향합니다.
