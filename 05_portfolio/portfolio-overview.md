# Portfolio Overview

이 저장소는 손기석의 엔지니어 포트폴리오를 면접관과 채용 담당자가 빠르게 이해할 수 있도록 정리한 문서입니다.

## 나는 어떤 엔지니어인가
손기석은 **백엔드 개발 경험**, **AI 응용서비스 구현 경험**, **정보보안 경력**을 함께 가진 엔지니어입니다.

단순 기능 구현자보다,
- 서비스 구조를 이해하고
- AI 기능을 서비스 흐름에 연결하며
- 운영 가능한 구조를 만들고
- 필요할 때 보안·통제 관점까지 같이 볼 수 있는

실무형 엔지니어를 지향합니다.

---

## 핵심 강점
### 1. 백엔드 중심 개발 경험
- Java, Spring Boot 기반 웹 애플리케이션 및 API 개발
- Oracle, PostgreSQL 기반 DB 연계 및 SQL 처리
- 운영 반영, 기능 개선, 장애 대응 경험

### 2. AI 응용서비스 경험
- LLM 챗봇 PoC
- Text2SQL 구조 구현
- 문서 기반 AI 서비스 구조 검토
- FastAPI 기반 AI 응용 API 설계/구현 경험

### 3. 운영/배포/환경 구성 경험
- Docker 기반 실행 환경 구성
- Linux 서버 및 원격 개발환경 운영
- Jenkins, Git 기반 협업 및 배포 구조 경험

### 4. 보안 도메인 차별화
- 정보보안 경력 보유
- 보안 점검, 운영 지원, 인증/통제 맥락 이해
- 보안과 백엔드를 연결할 수 있는 강점

---

## 주요 프로젝트 축
### 1. 데이터 분석 결과 서비스화
분석 결과를 실제 웹 서비스/API/업무 기능으로 연결하는 프로젝트

### 2. AI 응용 기능 접목
RAG, Text2SQL, 문서 기반 QA 등 LLM 기반 기능을 서비스에 접목하는 프로젝트

### 3. 운영형 구조 및 플랫폼 개선
운영 가능한 개발 구조, 배포 구조, 실행 환경, 협업 체계를 정리하는 프로젝트

### 4. 보안/권한/감사 관점 기능
보안 도메인과 연결되는 로그, 권한, 감사 기능을 기술적으로 구조화하는 프로젝트
