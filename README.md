# Engineering Career Portfolio

손기석의 엔지니어 포트폴리오 저장소입니다.

이 저장소는 면접관과 채용 담당자가 손기석의 기술 방향, 강점, 프로젝트 성격을 빠르게 이해할 수 있도록 구성합니다.

## 핵심 소개
- Java/Spring 기반 백엔드 개발 경험
- Python/FastAPI 기반 API 및 AI 응용서비스 경험
- 운영 반영, 실행 환경 구성, 장애 대응 경험
- 정보보안 경력 기반의 차별화 요소 보유

## 폴더를 3개로 나눈 이유
이 저장소는 **소개 → 증명 → 포지셔닝** 순서로 읽히도록 구성했습니다.

### 01_profile
이 폴더는 **"손기석이 어떤 엔지니어인지"** 소개하는 영역입니다.

예시:
- `career-summary.md` : 경력 전체 요약
- `core-strengths.md` : 핵심 강점 정리
- `career-direction.md` : 앞으로의 기술 방향 설명

즉, 면접관이 처음 들어왔을 때 **사람 소개**를 빠르게 파악하는 폴더입니다.

### 02_projects
이 폴더는 **"실제로 무엇을 만들고 구현할 수 있는지"** 보여주는 영역입니다.

예시:
- `ai-rag-api/README.md` : 문서 기반 질의응답 API 프로젝트
- `text2sql/README.md` : 자연어 질의 → SQL 변환 프로젝트
- `backend-platform-template/README.md` : 운영형 백엔드 템플릿
- `security-audit-log/README.md` : 감사로그/권한통제 예제

즉, 말로 소개한 역량을 실제 결과물로 보여주는 **기술 증명** 폴더입니다.

### 03_portfolio
이 폴더는 **"그래서 어떤 포지션에 적합한지"** 채용 관점에서 정리하는 영역입니다.

예시:
- `portfolio-overview.md` : 전체 포트폴리오 개요
- `backend-ai-engineer.md` : 백엔드 + AI 응용 엔지니어 포지셔닝
- `security-backend-platform.md` : 보안 + 백엔드 + 플랫폼 특화 포지셔닝

즉, 프로필과 프로젝트를 묶어서 채용 관점으로 해석해주는 **포지셔닝** 폴더입니다.

## 현재 포지셔닝
손기석은 **백엔드 개발을 중심으로 AI 응용을 확장하고, 데이터/플랫폼 역량을 함께 쌓으며, 보안 도메인을 차별화 포인트로 활용하는 엔지니어**를 지향합니다.
